# Capitalism At It's Finest

An absurdist capitalist-satire financial education web game.
Start with virtual money, gamble it away, go broke, get a job, repeat.

---

## Indítás — Magyar

### Windows (egyszerű)

Kattints duplán a **`start.bat`** fájlra — telepíti a függőségeket, elindítja a szervert és megnyitja a böngészőt automatikusan.

### Kézzel

1. **Függőségek telepítése:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Szerver indítása:**
   ```bash
   python app.py
   ```

3. **Megnyitás böngészőben:**
   ```
   http://localhost:5000
   ```

## Launch — English

### Windows (easy)

Double-click **`start.bat`** — it installs dependencies, starts the server, and opens the browser automatically.

### Manual

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Start the server:**
   ```bash
   python app.py
   ```

3. **Open in your browser:**
   ```
   http://localhost:5000
   ```
